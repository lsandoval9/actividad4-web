Tooling

Este proyecto está configurado y se compila con Angular CLI. Se recomienda utilizar VSCode + Angular Language Service para el desarrollo. Si necesitas probar tus componentes y páginas, puedes utilizar Jasmine y Karma (incluidos por defecto en Angular), o bien integrar otras herramientas como Cypress. Más instrucciones están disponibles en el archivo README.md.
